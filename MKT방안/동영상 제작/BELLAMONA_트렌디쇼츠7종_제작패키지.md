## ① 혈당 ASMR 실측쇼 「소리로 듣는 스파이크」 (50초)

제목: "마라탕 ASMR인데 왼쪽 위 숫자를 보세요"
캡션: "소리는 천국, 혈당은... 🌶️ #ASMR #마라탕 #혈당관리 #다낭성"
구조: 영상의 90%는 순수 먹방 ASMR. 유일한 이질 요소 = 좌상단 혈당 숫자(편집에서 합성).

씬 구성:
· 0-35초 [촬영] 마라탕/떡볶이 클로즈업 먹방 — 끓는 소리, 면 흡입, 씹는 소리 강조 (폰+무선이어폰 마이크로 충분). 편집 시 좌상단에 혈당 카운터 자막: 98 → 112 → 131 → 155 → 171 (5초 간격 상승, 효과음 없이 조용히)
· 35-42초 [Gemini G-1]
```
[공통 블록] 식탁에서 젓가락을 내려놓은 모나가 카메라를 정면으로 본다.
입가를 닦으며 담담하게: "맛있었어. 근데 방금 내 몸에서 무슨 일이 있었는지 볼래?"
```
· 42-50초 [촬영] 앱 혈당 그래프 화면녹화 풀샷 — 식사 시점 마커에서 치솟는 곡선. 자막: "소리는 천국이었지만... 저장하고 다음 편에서 '순서 바꾼 마라탕' 보기"
루프: 그래프 정점에서 컷 → 썸네일이 다시 마라탕이라 재생 반복 유도.
편집 노트: ASMR 파트에 자막·음악 절대 넣지 않기(이질감이 핵심 장치). 숫자만 조용히 오른다.

```

무전기를 잡고 벌떡 일어난다: "전군 주목. 연유라떼 유입 확인. 이건 훈련이 아니다."

씬3 [이어서] 부대장이 붉은 경보등 아래에서 화이트보드를 탁 치며:

씬4 [이어서] 다시 카페. 평상복 모나가 갑자기 멈칫하며 카메라를 본다:

씬5 [이어서] 관제실. 부대장이 땀을 닦으며 의자에 털썩:

씬6 [이어서] 부대장이 카메라에 바짝 다가와 진지하게:

```
편집: 씬4 뒤에 [촬영] 실제 그래프 3초 삽입("실화 데이터"). CTA 자막: "우리 부대 근무일지가 궁금하면 → 프로필 링크". 시리즈화: 야식편·회식편·공복운동편.
