집 거실의 따뜻한 자연광, 스마트폰 셀피 느낌의 캐주얼한 구도. 세로 사진.

## STEP 2. Gemini 생성 씬 6개 (프롬프트 전문 — 그대로 복사)

모든 프롬프트 실행 시 첨부: **P5 모나 이미지**. 씬별 표기된 촬영 소스도 함께 첨부.

### 공통 블록 (매 씬 프롬프트 맨 앞에 붙이기)
```
세로 9:16 영상. 한국 30대 여성 라이프스타일 브이로그. 스마트폰으로 찍은 듯한
핸드헬드 질감, 자연광, 밝고 따뜻한 톤. 등장인물은 첨부한 기준 이미지의 여성
'모나'로 얼굴·머리스타일을 일관되게 유지. 대사는 한국어, 밝지만 진솔한
30대 여성 목소리, 친구에게 말하듯 반말. 화면에 텍스트 오버레이는 넣지 말 것.
```

### G-1 콜드오픈 (8초)
```
[공통 블록] 저녁 거실 소파. 모나가 무릎을 안고 앉아 카메라를 물끄러미 보다가
작게 웃으며 말한다: "오늘 산부인과 다녀왔어. 다낭성난소증후군... 4년째래.
근데 오늘부터 나, 진짜 달라져 보려고." 표정은 지쳤지만 눈빛은 단단함.
```

### G-2 오프닝 시그니처 (8초)
```
[공통 블록] 낮의 밝은 거실. 모나가 카메라를 향해 활짝 웃으며 손을 흔든다:
"다낭성이랑 헤어지는 중! 모나 로그, 오늘이 D+7일차야. 일주일 전 얘기부터 해줄게."
경쾌한 에너지, 살짝 줌인되는 느낌.
```

### G-3 진료 회상 내레이션 씬 (8초)
```
[공통 블록] 카페 창가에 앉은 모나, 서류봉투를 내려다보다 카메라를 본다.
차분한 톤: "선생님이 그러더라. 약도 중요한데, 혈당 관리가 진짜 중요하다고.
다낭성이랑 인슐린이 그렇게 깊은 사이인 줄 몰랐어."
```

### G-4 센서 부착 리액션 (8초)
```
[공통 블록] 집 거실. 모나가 왼팔 상완에 붙은 동그란 흰색 센서를 신기한 듯
바라보다 카메라에 팔을 보여주며: "그래서 질렀다, 연속혈당측정기!
하나도 안 아팠어. 이제 내 혈당이 실시간으로 보인다는 거지?" 들뜬 표정.
```

### G-5 데이터 첫 반응 (8초)
```
[공통 블록] 소파에서 폰을 보던 모나의 눈이 커진다. 폰과 카메라를 번갈아 보며:
"잠깐만. 어제 먹은 라떼 하나에 혈당이 이렇게 뛴다고? 밥도 아니고 음료수인데?
야... 나 지금까지 뭘 모르고 살았던 거야." 충격과 흥미가 섞인 표정.
```

### G-6 클로징 (8초)
```
[공통 블록] 밤, 침대에 엎드려 턱을 괸 모나. 졸린 듯 편안한 미소:
"다음 주부터는 회사 다니면서 이걸 지킬 수 있는지 실험해볼 거야.
구내식당이 최대 고비겠지? D+14에서 만나. 잘 자." 손을 작게 흔들며 화면 어두워짐.
```
